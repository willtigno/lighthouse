--- OVERALL ---
The website is designed and coded in a MacBook Air (13-inch, Early 2015) with brackets.
Screen size and resolution: 13.3-inch (1440 x 900)

The website is divided into sections, with "row" being the parent followed by "large-11 large-centered" so the content doesn't take up the whole viewport.

A page must consist of a navigation bar and footer.
See template.html for reference.

It is highly recommmended to put all content inside the two divs.

Responsive navigation is based on Foundation document
http://foundation.zurb.com/sites/docs/responsive-navigation.html
